## 📂 About This Repository

This repository serves as my personal workspace and archive for exploring the C++ programming language. It contains a variety of mini-projects, algorithmic exercises, and console applications designed to solidify my understanding of computer science fundamentals.

Rather than a single large application, this is a living collection of code that grows as I learn and experiment with new concepts.

### 🧠 Core Themes Explored

While the programs vary in scope (typically under 200 lines of code), they generally cover the following areas:

<ul>
  <li><strong>Object-Oriented Programming (OOP):</strong> Classes, inheritance, polymorphism, and encapsulation.</li>
  <li><strong>Core Mechanics & Memory:</strong> Pointers, templates, bit manipulation and standard libraries.</li>
  <li><strong>Data Structures & Algorithms:</strong> Custom implementations and logic exercises.</li>
</ul>
